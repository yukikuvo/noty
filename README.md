# Installation guide

## Linux

* Download the compiled release 
* Build using Vala file:

```
$ valac --pkg gtk+-3.0 --pkg webkit2gtk-4.0 webkit-sample.vala
```

## IOS

* Download profile
* Install to settings
* Done
[Link](https://github.com/yukikuvo/noty/releases/)

## Windows and MacOS

Use the PWA version of our application

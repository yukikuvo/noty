# Noty PWA

## About

Noty - is beautiful app with user-friendly design made to keep your notes. 

[Main Site](https://yukikuvo.github.io/noty/pwa/index.html)

## Features

* Dark Mode
* Local Storage Database
* Clear all
* Export and import data
* Theme lists

#### To-do features
